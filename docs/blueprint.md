# **App Name**: Expo Camera App

## Core Features:

- Camera Preview: Display a live camera preview using react-native-vision-camera.
- Photo Capture: Capture photos with the camera using the vision camera library.
- Video Recording: Record videos with the camera using the vision camera library.
- QR/Barcode Scanning: Scan QR codes and barcodes in real-time using the camera feed.
- Text Translation: Use AI to translate Text from Images to different languages and write the translation to the camera view.

## Style Guidelines:

- Primary color: Deep sky blue (#43A1FF) to evoke a sense of clarity and precision.
- Background color: Light gray (#F0F4F7), a desaturated form of the primary color for a modern feel.
- Accent color: Lime green (#B1FF43) for highlights, particularly in AI translation related UI elements.
- Body and headline font: 'Inter', a sans-serif font known for its modern and neutral appearance, suitable for both headlines and body text.
- Use minimalistic icons for camera controls and settings.
- Clean and intuitive layout with camera controls easily accessible.
- Smooth transitions for UI elements and feedback for user interactions.
